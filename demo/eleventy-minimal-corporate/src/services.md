## Process

1. **Review Figma designs** - Analyze layouts, spacing, typography
2. **Build Eleventy structure** - Clean folder organization
3. **Implement pages** - Semantic HTML + minimal CSS
4. **Test performance** - Lighthouse, accessibility, cross-browser
5. **Deploy to Cloudflare** - Connect Git, configure build
6. **Handoff** - Documentation + training

**Timeline:** 1-3 weeks depending on site complexity
